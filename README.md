# fd-clone-website

A full-stack Facebook-inspired social media application built to replicate many of the core features of Facebook while serving as a learning project for modern web development. The project focuses on creating a responsive, user-friendly platform where users can connect, share content, and interact with others in real time.

## 📌 Project Overview

The goal of this project is to develop a clone of Facebook that demonstrates the implementation of authentication, user management, social interactions, and responsive UI design. It follows modern web development practices with a scalable architecture and clean code structure.

This project is intended for educational purposes and is not affiliated with or endorsed by Meta Platforms, Inc.

## ✨ Features

- User authentication (Sign Up, Login, Logout)
- Secure password hashing and authentication
- User profile creation and management
- Create, edit, and delete posts
- Like and comment on posts
- News feed displaying posts from users
- Upload profile pictures and images
- Friend request system
- Responsive design for desktop and mobile devices
- Real-time updates (if implemented)
- Search users
- Notifications (if implemented)
